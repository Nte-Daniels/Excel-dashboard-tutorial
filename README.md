# Excel-dashboard-tutorial - resources
This repository contains all the materials used in my Excel tutorial on YouTube.

## What's Inside
- **dataset/** – Raw and cleaned datasets used in the project  
- **pbix/** – Final Excel dashboard file  
- **images/** – Supporting screenshots and graphics  
- **docs/** – Extra notes (if any)

## Requirements
- Excel or any tool that can open CSV files

## How to Use
1. Download the entire repository (top right → Code → Download ZIP).
2. Open the Excel file.
3. Explore, modify, or recreate the visuals as shown in the tutorial.
